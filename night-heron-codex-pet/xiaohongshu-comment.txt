夜鹭 Codex 宠物下载方式：

1. 打开 GitHub 仓库
2. 点绿色 Code 按钮
3. 点 Download ZIP
4. 解压后，把里面的 night-heron 文件夹复制到 ~/.codex/pets/
5. 完全退出并重启 Codex
6. 到 Settings → Pets 里选择「夜鹭」

注意：最终路径要是 ~/.codex/pets/night-heron/pet.json，不要多套一层仓库文件夹。

如果没有显示，可以在 ~/.codex/config.toml 的 [desktop] 下加：

selected-avatar-id = "custom:night-heron"
